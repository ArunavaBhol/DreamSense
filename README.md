# 🧠 DreamSense: EEG-Based Dream Emotion Analysis

## 📌 Overview
DreamSense is an AI-based system that analyzes EEG signals during sleep to interpret underlying emotional states. Instead of reconstructing exact dreams, it focuses on extracting patterns and representing them as emotion-based insights.

---

## 🎯 Objectives
- Analyze EEG sleep data  
- Extract meaningful brain signal features  
- Identify patterns related to emotional states  
- Build a foundation for future dream visualization  

---

## ⚙️ Current Progress
- EEG Data Acquisition (Sleep-EDF Dataset)  
- Signal Preprocessing (Filtering & Noise Removal)  
- Feature Extraction (Frequency bands, statistical features)  
- Exploratory Data Analysis  

---

## 🔬 Methodology
- EEG signal preprocessing  
- Feature extraction (Delta, Theta, Alpha, Beta bands)  
- Statistical analysis of brain activity patterns  

---

## 🛠️ Tech Stack
- Python  
- NumPy, Pandas  
- MNE-Python  
- Matplotlib / Seaborn  
- Scikit-learn (planned)  

---

## 🚀 Future Work
- Machine learning-based emotion classification  
- AI-based dream visualization  
- Real-time EEG integration  

---

## ⚠️ Note
DreamSense does not reconstruct exact dreams. It focuses on interpreting emotional patterns from EEG signals.
